关于对Hexo制作Blog的更新问题

Created at 2017-02-09 Updated at 2017-02-09 Category Hexo Tag technology

部署主题，本地查看效果

返回Hexo目录，右键git bash，输入

hexo g
hexo s

打开浏览器，输入http://localhost:4000/ 及预览本地主题更换




如果效果满意,将它部署到github上

打开Hexo文件夹，右键git bash，输入

hexo clean （必须要，不然有时因为缓存问题，服务器更新不了主题）
hexo g -d



打开自己主页即可以浏览自己Blog效果
